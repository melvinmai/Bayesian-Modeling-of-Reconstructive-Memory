This is a Python implementation of the Bayesian Model of Reconstructive Memory 
described in Hemmer, P. & Steyvers, M. (2009) paper called "A Bayesian account of Reconstructive Memory"

Referenced paper:
Hemmer, P. and Steyvers, M. (2009), A Bayesian Account of Reconstructive Memory. Topics in Cognitive Science, 1: 189-202. https://doi.org/10.1111/j.1756-8765.2008.01010.x
